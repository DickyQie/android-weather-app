###	Android------个人项目（歆语气象通）

<p>Android MVP+Retrofit+Rxjava 开发</p>
<p>歆语气象通： 歆语气象伴随你的身边，便捷生活。</p> 
<p>包含了以下功能：</p> 
<p>1. 天气预报数据覆盖中国城市和地区；<br> 2. 提供一周天气预报及最低最高温度，时刻关注天气，轻松计划出行；<br> 3. 各种指数详细信息，如太阳，运动等。<br> 4.城市查询：提供中国多个市，县，自治区等的查询。<br> <br> &nbsp;</p>
